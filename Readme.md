Bill

Bill egy számlázó és leltárkarbantartó program. 
Az adatbázisban tároljuk a leltárat. Egy termék nevét, árát, leírását és a rendelkezésre álló mennyiséget.
Lehet újakat felvenni és a meglév?eket módosítani. Mindezt egy Swinges felületen.
A program lényege ,hogy egy szimpla gombnyomással a termékeinkb?l kiválasztottat hozzá tudjuk adni a számlához amit egy táblázat személyesít meg.
A program számon tartja az aktuális végösszeget. Természetesen kivenni is lehet a már berakott tételeket.
Illetve ellehet menteni egy xml formátumban és ezeket a program kezeli és vissza iis képes tölteni. Ezután a módosítás természetesen már nem lehetséges.
